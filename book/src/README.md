# centurion

**centurion** is an open-source desktop application that leverages the IEEE 1278.1-2012
Distributed Interactive Simulation (DIS) application protocol for performing simulation
management (SIMAN) operations as defined in the aforementioned standard.

## Contributing

Centurion is free and open source. You can find the source code as well as
report issues and create feature requests on [GitHub](https://github.com/crhowell3/centurion).
